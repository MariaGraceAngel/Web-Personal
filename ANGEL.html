HTML


Internal Style

HTML (Hypertext Markup Language) adalah bahasa markup standar yang digunakan untuk membuat dan mendesain halaman web. HTML digunakan untuk struktur dasar sebuah halaman web, menentukan elemen-elemen seperti teks, gambar, tautan, dan elemen-elemen lainnya yang membangun konten halaman web. Berikut adalah beberapa konsep dasar HTML:

Elemen: Setiap bagian dari sebuah halaman web dibungkus dalam elemen HTML. Elemen HTML biasanya terdiri dari tag pembuka, konten, dan tag penutup.

Tag: Tag HTML digunakan untuk menentukan jenis elemen dan bagaimana elemen tersebut akan ditampilkan oleh browser web. Tag terdiri dari karakter < dan > yang mengapit nama elemen. Ada tag pembuka (opening tag) dan tag penutup (closing tag).

Atribut: Atribut memberikan informasi tambahan tentang elemen tertentu. Mereka ditambahkan ke dalam tag pembuka dan digunakan untuk mengatur properti atau perilaku elemen tersebut. Contoh atribut adalah src untuk menentukan sumber gambar atau href untuk menautkan ke halaman web lain.

Struktur Dasar: Sebuah dokumen HTML dimulai dengan tag yang menandakan awal dari dokumen, diikuti oleh yang berisi informasi meta tentang dokumen, seperti judul halaman, dan yang berisi konten yang akan ditampilkan di browser.

Teks: HTML digunakan untuk menampilkan teks dalam berbagai format, termasuk paragraf, heading (judul), daftar, dan teks berformating seperti tebal atau miring.

Gambar dan Media: HTML memungkinkan penambahan gambar, audio, dan video ke dalam halaman web menggunakan elemen khusus.

Tautan: Tautan atau hyperlink digunakan untuk menghubungkan halaman web satu dengan yang lain.

Komentar: Komentar tidak akan ditampilkan di browser, tetapi berguna untuk menjelaskan bagian-bagian tertentu dari kode kepada pembaca atau pengembang lainnya.

HTML adalah bagian penting dari pengembangan web karena menyediakan struktur dasar yang diperlukan untuk menampilkan konten di browser web. Itu merupakan fondasi bagi teknologi web modern seperti CSS (Cascading Style Sheets) untuk tata letak dan styling, serta JavaScript untuk interaksi dinamis dan fungsionalitas tambahan.
